Вариант 29. Определить структурный тип, описывающий маркетплейсы (название маркетплейса, количество ПВЗ по стране, город дислокации, рыночная стоимость, тип маркетплейса: продуктовый, товары для дома, все-подрял; тип бизнеса компании: основной, вспомогательный, франшиза и др.). В отдельный массив поместить все маркетплейсы не из Казани. Реализовать вывод отфильтрованных данных в виде оберточной функции.
